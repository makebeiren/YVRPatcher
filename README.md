# YVRPatcher
﻿
YVRPatcher是一款基于QuestPatcher图形用户界面的模组安装工具，本版本为YVR的beatsaber进行了适配
﻿
它支持使用[QuestLoader](https://github.com/sc2ad/QuestLoader/)和[Scotland2](https://github.com/sc2ad/Scotland2)这两个模组加载器进行模组化。
QuestPatcher所使用的QMOD格式在[此处](https://github.com/Lauriethefish/QuestPatcher.QMod/tree/main/SPECIFICATION.md)有详细说明。

# YVRPatcher-machine translation

YVRPatcher is a mod installation tool based on the QuestPatcher graphical user interface, and this version has been adapted for YVR's Beatsaber

It supports modding using two mod loaders: [QuestLoader](https://github.com/sc2ad/QuestLoader/) and [Scotland2](https://github.com/sc2ad/Scotland2).
The QMOD format used by QuestPatcher is detailed in [this location](https://github.com/Lauriethefish/QuestPatcher.QMod/tree/main/SPECIFICATION.md).
