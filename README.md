# YVRPatcher
YVRPatcher是一款基于QuestPatcher图形用户界面的模组安装工具，本版本为YVR的Beat Saber进行了适配。

它支持两款模组加载器完成游戏模组安装。
QuestPatcher所使用的QMOD格式可在对应文档中查看详细说明。

---

# YVRPatcher (English)
YVRPatcher is a mod installation tool based on the QuestPatcher graphical user interface, this version is adapted for Beat Saber on YVR.

It supports modding with two mod loaders.
The QMOD format used by QuestPatcher is detailed in the official document.
